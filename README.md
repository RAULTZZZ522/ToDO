# 微信小程序云开发示例项目

## 项目说明
这是微信小程序云开发的基础框架示例，包含以下功能：

- 云函数：调用云函数示例
- 数据库：数据库操作示例
- 存储：文件上传示例

## 目录结构

```
- cloudfunctions/ 云函数目录
  - login/ 云函数-登录
- miniprogram/ 小程序主目录 
  - pages/ 页面文件
  - style/ 样式
  - app.js 项目入口文件
  - app.json 全局配置
```

## 使用说明

1. 在微信开发者工具中打开项目
2. 开通云开发功能
3. 创建云环境
4. 修改app.js中的环境ID
5. 上传部署云函数
